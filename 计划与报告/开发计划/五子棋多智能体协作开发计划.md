# 五子棋多智能体协作系统 开发计划

> 基于 LangGraph 最新版，遵循 State / Nodes / Edges 核心架构。

---

## 一、LangGraph 架构与设计原则

| LangGraph 概念 | 本系统映射 |
|---|---|
| **State** | `MultiAgentState`(TypedDict) — Node 间流转的共享状态，含 game_report、各 Agent 输出、对话历史 |
| **Nodes** | 每个处理步骤 = 一个 Python 函数，签名 `(State) → dict`，返回部分 State 更新 |
| **Edges** | 普通边 `START → extract → phase_check`；条件边 `route_by_phase()` 根据局势分级路由 |
| **StateGraph** | `StateGraph(MultiAgentState)` 构建器，注册 Node & Edge，编译为可执行 Graph |
| **MemorySaver** | LangGraph 内置 checkpointer，持久化图状态，支持反问后恢复 |

**设计原则**：
- **OOP**：一个文件定义一个核心类，封装行为，通过接口交互
- **单一职责**：棋盘编解码 ≠ LLM调用 ≠ 信息提取 ≠ Agent逻辑
- **高内聚低耦合**：同类功能同文件夹；Agent 之间互不知晓，只通过 State 传递信息；Orchestrator 是唯一聚合点
- **API 一致**：所有 LLM 调用严格沿用 `openai.OpenAI` + DashScope endpoint + `DASHSCOPE_API_KEY`

### 3.3 `core/` — 编排核心

**`protocol.py`** — 消息数据结构（纯 dataclass，无行为）：
- `Proposal`: agent_role, move(x,y), reasoning, confidence
- `Critique`: target_move, concern, severity
- `FinalDecision`: move, reason, agent_summaries

**`state.py`** — LangGraph State 定义：

```python
class MultiAgentState(TypedDict):
    game_report: str              # GameInfoExtractor 输出
    turn_count: int
    current_color: str            # 'X' / 'O'
    tactical_proposal: Optional[dict]
    defense_proposal: Optional[dict]
    devil_critiques: Optional[list]
    chief_decision: Optional[dict]
    messages: Annotated[list[BaseMessage], add_messages]   # reducer 追加
    human_question: str           # "" 表示无反问
    is_rethinking: bool
    phase: str                    # emergency / simple / normal / complex
    skip_llm: bool
```

**`orchestrator.py`** — 图构建与编排：

```python
class MultiAgentOrchestrator:
    def __init__(self, llm_client):
        self._agents = { "tactical": ..., "defense": ..., "devil": ..., "chief": ... }
        self._graph = self._build_graph()

    def analyze(self, controller, human_question="") -> FinalDecision:
        state = {"game_report": GameInfoExtractor(controller).build_game_report(), ...}
        return self._graph.invoke(state)

    # ---- Node 函数 ----
    def _phase_check_node(self, state) -> dict:
        phase = self._speed_ctrl.classify_phase(...)
        return {"phase": phase, "skip_llm": phase in ("emergency","simple")}

    def _tactical_node(self, state) -> dict:
        return {"tactical_proposal": self._agents["tactical"].think(state["game_report"])}

    def _defense_node(self, state) -> dict:
        return {"defense_proposal": self._agents["defense"].think(state["game_report"])}

    def _devil_advocate_node(self, state) -> dict:
        return {"devil_critiques": self._agents["devil"].think(...)}

    def _chief_node(self, state) -> dict:
        return {"chief_decision": self._agents["chief"].think(...)}

    # ---- 条件路由 ----
    def _route_by_phase(self, state) -> str:
        if state.get("skip_llm"): return "skip"
        if state["phase"] == "complex": return "full"
        return "quick"

    # ---- 图构建 ----
    def _build_graph(self):
        builder = StateGraph(MultiAgentState)
        builder.add_node("phase_check", self._phase_check_node)
        builder.add_node("engine_fallback", ...)
        builder.add_node("tactical", self._tactical_node)
        builder.add_node("defense", self._defense_node)
        builder.add_node("devil_advocate", self._devil_advocate_node)
        builder.add_node("chief", self._chief_node)

        builder.add_edge(START, "phase_check")
        builder.add_edge("engine_fallback", END)
        builder.add_conditional_edges("phase_check", self._route_by_phase, {
            "skip": "engine_fallback", "quick": "tactical", "full": "tactical"})
        builder.add_edge("tactical", "defense")
        builder.add_edge("defense", "chief")
        builder.add_edge("chief", END)
        return builder.compile(checkpointer=MemorySaver())
```

### 3.4 `agents/` — 四个 Agent

| Agent | 角色 | 输入 | 输出 |
|---|---|---|---|
| **TacticalAnalyst** | 找进攻机会 + 结合**己方技能**调整 | game_report（含己方技能段落） | Proposal（1-2个候选 + 理由 + 置信度） |
| **DefenseSpecialist** | 找对手威胁 + 结合**敌方技能**预警 | game_report（含敌方技能段落 + 技能子位置 + 封锁位） | Proposal（防守候选 + 风险预警） |
| **DevilAdvocate** | 对 Tactical + Defense 的候选找漏洞 | 前两个 Proposal + game_report | List[Critique]（最多2条） |
| **ChiefStrategist** | 综合所有意见，做最终裁决 | 全部 Proposal + Critique + game_report | FinalDecision（最终落子 + 理由） |

**`base_agent.py`** 模板方法：

```python
class BaseAgent(ABC):
    role_name: str           # abstract property
    system_prompt: str       # abstract property

    def think(self, game_report: str, **context):
        user_msg = self.build_user_message(game_report, **context)
        raw = self._llm.chat(self.system_prompt, user_msg)
        return self.parse_response(raw)

    @abstractmethod
    def build_user_message(self, game_report, **context) -> str: ...
    @abstractmethod
    def parse_response(self, text: str): ...
```

**战术官/防守官技能考量**：

| 己方技能（战术官利用） | 敌方技能（防守官预警） |
|---|---|
| 万宁阵法(每3回+1) → 落子更大胆 | 克敌先机(对手连4补子) → 4连不保险，需双杀 |
| 四方阵(斜向+1) → 优先选斜向有潜力的位置 | 八卦阵(偷边角) → 关键棋不放边角 |
| 血狱影杀阵(20%邻位+1) → 优先在密集区落子 | 绝户阵(限我3技能子) → 技能子不超3 |
| 归元阵(连4补子) → 积极构建接近4连 | 困龙阵(同线封锁) → 避免连续同线落子 |
| 梅花阵(花苞) → 用花苞卡对方路线 | 梅花阵(花苞) → 检查花苞位置，不要踩中 |
| 绝户阵(限敌3技能子) → 放心用主动技能 | 五雷阵(清技能子) → 技能子不放要害位 |

### 3.5 `game_info/extractor.py` — GameInfoExtractor

**唯一** 直接访问 `GameController` 的类。从 controller 提取全部信息，生成 `game_report` 文本供所有 Agent 共用。

```
controller.board.grid  ──→  board_codec.board_to_text()
controller.turn_count
controller.current_player.skill.skill_name / description / can_activate()
controller.opponent_player.skill.skill_name / description
controller.board.find_lines(color, n)  ──→  威胁分析文本
controller.board.count_skill_stones_of(color)
controller.board.get_skill_stone_positions_of(color)
controller.board.is_blocked(x, y)  (遍历225格)
        │
        ▼
  build_game_report() → str
  (含: 回合信息 / 棋盘 / 威胁 / 己方技能 / 敌方技能 / Top-N候选)
```

### 3.6 其余模块

- **`SlidingMemory`**：deque 存储最近 10 步决策摘要 + 5 条关键事件，提供 `get_context_for_prompt(max_tokens)` 用于注入 Agent 上下文
- **`SpeedController`**：局势分级（emergency/simple → 跳过 LLM；normal → 跳过反对官；complex → 全流程）+ 超时熔断（单 Agent 3s / 总流程 8s）
- **`PromptLoader`**：启动时加载 `prompts/*.txt` 到内存，按角色名提供 `get("tactical")`

from openai import OpenAI
client = OpenAI(
    api_key=os.getenv("DASHSCOPE_API_KEY"),
    base_url="https://dashscope.aliyuncs.com/compatible-mode/v1")

response = client.chat.completions.create(
    model="qwen-turbo",           # 战术/防守用 turbo，反对/chief 用 plus
    messages=[{"role":"system","content":system_prompt},
              {"role":"user","content":user_message}],
    temperature=0.1)

content = response.choices[0].message.content
