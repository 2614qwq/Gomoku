五子棋获胜常见条件
==================

本文档列举五子棋（连珠）中所有决定胜负的棋形与定式，供智能体在进攻/防守决策时参考。

---

一、终局获胜条件（满足其一即胜）

1. 五连
   五颗同色棋子横、竖、斜连成一线，直接获胜。

2. 活四（连四）
   四颗棋子一线，两端均为空位。对手无论堵哪端，下一步都能五连。
   → 活四形成 = 立即获胜（对手无法阻挡）

3. 四三杀（四三）
   一颗棋子同时制造"一个四（活四或冲四）"和"一个活三"。
   对手堵四 → 活三变活四 → 五连胜；对手堵三 → 四变五胜。
   → 四三杀形成 = 必胜（对手无论堵哪边都挡不住）

4. 双活三杀（三三）
   一颗棋子同时制造两个"活三"。
   对手只能堵一个活三 → 另一个活三变活四 → 五连胜。
   → 双活三形成 = 必胜

5. 四四杀（四四）
   一颗棋子同时制造两个"四"（至少一个为活四）。
   对手只能堵一个四 → 另一个变五连。
   → 四四形成 = 必胜（需注意至少一方为活四，否则可能被反杀）

6. 连续追胜（VCF / VCT）
   通过一连串冲四（VCF）或混合威胁（VCT），步步迫使对手被动防守，
   最终走到活四或五连。整个过程对手没有反攻机会。

---

二、进攻构造材料

1. 活三
   三颗棋子一线，两端均空，下一步可形成活四。
   是最基础的进攻单元，绝大多数必胜定式依赖活三。

2. 冲四（死四）
   四颗棋子一线，仅一端空。对手必须堵那一端。
   冲四本身不直接赢，但可以：
   - 配合其他位置的活三形成四三杀
   - 作为 VCF 链条中的一环

3. 眠三（死三）
   三颗棋子一线，仅一端可延伸为冲四（另一端被堵或到边）。
   可用于辅助构造，但不直接构成致命威胁。

4. 活二
   两颗棋子一线，两端均空，可发展为活三。
   **活二是所有进攻的起点。** 大量构造活二是制造复杂进攻的基础。
   常见的活二构型：
   - 跳活二：两子间隔一格（如 (7,7) 和 (9,7)，中间 (8,7) 空）
   - 连活二：两子相邻（如 (7,7) 和 (8,7)）

5. 双活二交叉（梅花型）
   多个活二在不同方向交叉于一点，该点落子可同时激活多个活三。
   是构造双活三杀和四三杀的核心手法。

---

三、进攻构造策略（战术官参考）

1. 活二 → 活三 → 四三杀 / 双活三 → 五连
   这是最经典的渐进式进攻路线。优先积累活二，然后寻找交叉点。

2. 一子多杀
   寻找能同时创造多个威胁的交叉点：
   - 一子同时形成活三 + 冲四（四三杀）
   - 一子同时形成两个活三（双活三杀）
   - 一子同时形成两个四（四四杀）

3. 连续冲四追胜（VCF）
   当拥有眠三或冲四时，计算连续的冲四路径，
   确保每一步都是冲四（对手必须应），最终导向活四或五连。

4. 梅花桩布局
   在中心区域以 L 形或十字形布局活二，
   形成多点开花局面，使对手难以同时防守。

5. 双线/多线进攻
   同时在两个方向上构造威胁（如横线活三 + 斜线冲四倾向），
   对手顾此失彼。

---

四、防守原则（防守官参考）

1. 防守优先级
   堵四三杀 > 堵双活三 > 堵活三 > 堵冲四 > 破坏活二

2. 提前破坏活二
   **最好的防守是让对手根本形不成活三。**
   当对手构造活二时，在活二的延伸方向抢占关键点，阻止其发展为活三。

3. 抢占交叉点
   当对手有多个活二指向同一位置时，抢先占据该交叉点，
   阻止对手一子多杀。

4. 堵活三的正确方式
   - 堵活三时选择对自己进攻有利的一端（能同时构造己方活二的优先）
   - 不要只被动堵，要"以攻为守"

5. 反杀优先级
   当己方也有进攻时，优先考虑能否"先一步胜"（比对手更早形成必胜定式）。

---

五、智能体决策要点

1. 即使已形成三三/四三杀等必胜定式，仍然必须实际将棋子连成五连。
   → 形成定式后，利用对手被迫防守的间隙，将活三推进为活四，最后五连。

2. 进攻时始终扫描棋盘上的活二数量、交叉点位置，
   选择能最大化活二→活三转化效率的落子。

3. 防守时不要被动挨打，要寻找"以攻代守"的机会。
   堵对手的同时，如果能在同一位置形成己方的活二或活三，则为最优解。

4. 中心区域（5x5 到 11x11）是活二/活三的主要战场，
   边角区域很难构造多方向威胁，不要主动去边角落子。

5. 所有进攻/防守判断必须优先于技能使用。
   先确定最佳落子位置，再考虑技能能否进一步提升效果。
