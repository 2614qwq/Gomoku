# 棋谱处理工具 —— 任务计划

**目标**：调用视觉大模型（qwen3-vl-plus）分析棋谱图片，将棋局转为 15x15 二维数组存入 CSV，为后续 RAG 检索提供数据基础。

---

## 1. 数据格式设计

一个棋局 = 一个 15x15 二维数组，CSV 每行是一个完整棋局。

```
0   = 空位
1   = 黑棋第1手（先手）
2   = 白棋第1手（后手）
3   = 黑棋第2手
...
N(奇数) = 黑棋   N(偶数) = 白棋
```

### CSV 表结构（3 列）

| board_name | source_image | grid |
|------------|-------------|------|
| 浦月开局 | img1.png | `[[0,0,...],[0,1,2,3,...],...]` |

文件：`棋谱csv数据库/chess_records.csv`
